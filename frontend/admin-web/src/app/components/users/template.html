User Management Guide

  • Create User: Go to Users → Add User → Fill details → Save
  • Assign Roles: Go to User → Role Mappings → Assign realm roles
  • Reset Password: Go to User → Credentials → Reset Password
  • Enable/Disable: Go to User → Settings → Toggle Enabled
  • View Sessions: Go to User → Sessions
